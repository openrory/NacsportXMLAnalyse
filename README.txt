-- INLEIDING --
Deze applicatie maakt een automatische analyse van het XML-bestand zoals dat door Nacsport wordt gegenereerd.

-- REQUIREMENTS --
Om deze applicatie te starten dient u aan de volgende requriements te voldoen:

- Python 3.5
- Python package xmltodict: gebruik "pip3 install xmltodict" om deze te installeren

-- STARTEN --
U roept het bestand aan op de volgende manier: python3 xml_analyse.py "<pad+bestandsnaam>" "<naam outputbestand>"}
